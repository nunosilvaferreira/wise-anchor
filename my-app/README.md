# WiseAnchor

## Overview

As a software developer, I am interested in building tools that help people create structure in their daily lives. Clear routines and predictable transitions between activities can make a big difference in productivity and wellbeing.

WiseAnchor is a simple web application designed to help users manage daily routines and track progress through tasks. The long-term vision of this project is to create a tool that can guide users step-by-step through their day and help them move smoothly from one activity to the next.

This first version of WiseAnchor is a small web application built using Next.js. The current implementation demonstrates a basic application structure and displays a simple greeting message to confirm that the development environment is working correctly.

Even though the functionality is simple, the purpose of this project is to establish a clean development workflow using modern web tools such as Next.js, Git, and GitHub. This foundation will allow the project to grow into a more complete routine management application in the future.

## Development Environment

The software was developed using the following tools and technologies:

- Visual Studio Code
- Git
- GitHub
- Node.js
- Next.js

The programming language used in this project is **JavaScript**, and the application is built using the **Next.js framework**, which allows developers to create modern web applications with React and server-side capabilities.

The application runs locally using the Next.js development server.
